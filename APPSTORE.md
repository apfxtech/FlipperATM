# ATM

ATM is a music player for Flipper Zero that plays 4-channel 8-bit audio from a text-based tracker format (.atm).

Songs are written as readable text, parsed on-device, and played through the built-in ATM-style synth engine.

Features:
- 4-channel 8-bit playback
- Text format (ATM1) for easy editing
- Real-time channel level meters
- Play/Pause, Stop, and volume control
- Fast loading from app assets
